# Safety Machinery Inspection

Aplikasi **Safety Machinery Inspection** adalah sistem monitoring dan inspeksi keselamatan mesin berbasis web, dibangun menggunakan Laravel 12.

> Untuk melihat versi full dan mendapatkan akses lengkap, silahkan hubungi saya.

---

## Fitur

- **Dashboard** — Ringkasan data mesin per area (FA, FB, FC, MTC)
- **Data Mesin** — CRUD data mesin dengan detail inspeksi keselamatan
- **OPL** — Data Operation Procedure Leader
- **OMOB** — Data dokumen OMOB
- **Agenda** — Kalender jadwal inspeksi
- **Scan** — Pencarian mesin via scan/nomor mesin
- **Riwayat** — Riwayat inspeksi per tanggal
- **Laporan** — Export laporan ke Excel & PDF
- **Manajemen Pengguna** — Level admin & user
- **Lokasi & Departemen** — Pengaturan area kerja

## Tech Stack

- Laravel 12
- PHP 8.2+
- MySQL / PostgreSQL
- Bootstrap 4
- SweetAlert2
- FullCalendar
- Font Awesome
- DomPDF
- PhpSpreadsheet
- Barcode Generator
